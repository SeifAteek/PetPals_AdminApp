Flash drive tip
---------------
Copy the whole Petpals folder to your USB drive, or copy only the four .exe
files listed above from each app folder.
Each .exe is about 85-95 MB (normal for desktop apps with a built-in browser).

Windows only — these are Windows portable executables (.exe).


To rebuild the .exe files (developers only)
-------------------------------------------
  cd <app-folder>
  npm install
  npm run electron:build:win
